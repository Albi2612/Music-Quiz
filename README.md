🧠 About the Project

This project is a lightweight quiz platform built to help users practice core music theory concepts in a fun and engaging way. 
Whether you're just starting out or brushing up your fundamentals, the app gives you quick questions and instant feedback.

I built this as a hands-on way to combine my interest in music with web development and backend logic.

🛠️ Tech Stack
Frontend: HTML
Backend: Python (Flask)
Deployment: Render
✨ Features
Multiple-choice music theory questions
Instant feedback on answers
Simple and clean UI
Fast and lightweight (no unnecessary complexity)
📦 Installation (Run Locally)

Clone the repository:

git clone https://github.com/your-username/music-quiz.git
cd music-quiz

Create a virtual environment (optional but recommended):

python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate

Install dependencies:

pip install flask

Run the app:

python app.py

Open your browser and go to:

http://127.0.0.1:5000/
